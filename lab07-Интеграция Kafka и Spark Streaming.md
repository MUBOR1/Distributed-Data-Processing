### 4.2 Spark Streaming Application (***spark_streaming.py***)
**–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ:** –ß—Ç–µ–Ω–∏–µ –∏ –æ–±—Ä–∞–±–æ—Ç–∫–∞ –ø–æ—Ç–æ–∫–æ–≤—ã—Ö –¥–∞–Ω–Ω—ã—Ö –∏–∑ Kafka –≤ —Ä–µ–∞–ª—å–Ω–æ–º –≤—Ä–µ–º–µ–Ω–∏.

```python

spark = SparkSession.builder \
    .appName("KafkaSparkIntegration") \
    .master("local[2]") \
    .config("spark.jars", "/home/clay/kafka_lab7/jars/spark-sql-kafka-0-10_2.12-3.5.0.jar") \
    .getOrCreate()

# –ß—Ç–µ–Ω–∏–µ –¥–∞–Ω–Ω—ã—Ö –∏–∑ Kafka
kafka_df = spark.readStream \
    .format("kafka") \
    .option("kafka.bootstrap.servers", "localhost:9092") \
    .option("subscribe", "real-time-transactions") \
    .option("startingOffsets", "latest") \
    .load()

# –°—Ö–µ–º–∞ –¥–∞–Ω–Ω—ã—Ö
schema = StructType([
    StructField("transaction_id", StringType()),
    StructField("user_id", StringType()),
    StructField("amount", DoubleType()),
    StructField("type", StringType()),
    StructField("timestamp", StringType()),
    StructField("location", StringType())
])

# –ü–∞—Ä—Å–∏–Ω–≥ JSON –∏ –≤—ã–≤–æ–¥
transactions = kafka_df.select(
    from_json(col("value").cast("string"), schema).alias("data")
).select("data.*")

# –ó–∞–ø—É—Å–∫ —Å—Ç—Ä–∏–º–∏–Ω–≥–∞
query = transactions.writeStream \
    .outputMode("append") \
    .format("console") \
    .option("truncate", "false") \
    .trigger(processingTime="10 seconds") \
    .start()
```
